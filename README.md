## react-redux-app-cli
A CLI for scaffolding React.js projects.

<br />

## Installation
`$ npm install -g react-redux-app-cli`

<br />

## Usage
`$ react init`

<br />

## Features
**Javascript**
- Language: ES6
- Loader: Babel
- Framework: React + Redux

**CSS/Preprocessors**
- Stylus
- Postcss
- Normalize.css

**Module loader**
- Webpack

**Package manage**
- npm

**Others**
- Eslint
